# 🐧 Penguin Coder | Career Intelligence Platform

> **Democratizing Career Intelligence for Students & Engineers**
> 
> Your AI-powered career compass for the Indian tech job market.

## 🎯 What is Penguin Coder?

Penguin Coder is a high-performance career intelligence platform that transforms career planning from guesswork into data-driven decision-making. Whether you're exploring a new role, evaluating career pivots, or building a learning roadmap, Penguin Coder synthesizes market data, AI insights, and ROI analysis to guide your next move.

**Built specifically for the Indian job market**, with salary insights in ₹ LPA, structured pathways, and real-time market intelligence.

---

## ✨ Core Features

### 🔍 **Heuristic Career Optimizer**
- Search any job role and instantly get market insights
- Fuzzy-matching algorithm finds similar roles and career paths
- Real-time salary data (₹ LPA) with trend analysis
- Discover hidden career transitions with ROI scores

### 🤖 **AI Roadmap Generator** (Powered by Google Gemini)
- Generate custom 5-stage learning syllabus in seconds
- AI-tailored for your target role and current skillset
- Step-by-step progression from beginner to professional
- Includes resources, timelines, and difficulty assessments

### 📊 **ROI Intelligence Matrix**
- Compare career options side-by-side
- Evaluate: Salary potential vs. Effort required vs. Timeline
- Auto-sorted rankings by your chosen metric (highest ROI, fastest path, etc.)
- Visual comparison charts for quick decision-making

### 💬 **Smart Chat Oracle**
- AI counselor for personalized career & academic advice
- Ask industry questions: "Is cloud architecture worth learning in 2026?"
- Get market-specific guidance: "What's the scope for DevOps in India?"
- Real-time answers based on current market data

### 🎨 **Performance-Driven UI**
- "Liquid Glass" theme for modern, intuitive experience
- Optimized for speed and readability
- Fully responsive—works seamlessly on mobile, tablet, desktop
- Zero-dependency frontend (no npm, no build tools)

## 🚀 Quick Start

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for AI features)
- Free Google Gemini API key (optional, for AI features)

### Installation & Usage

1. **Clone the repository**
   ```bash
   git clone https://github.com/Penguin-SAEZO/penguin-career-matrix.git
   cd penguin-career-matrix
   ```

2. **Open in browser**
   ```bash
   # Simply open the file
   open index.html
   # or
   xdg-open index.html  # Linux
   ```

3. **Enable AI Features (Optional)**
   - Go to [Google AI Studio](https://aistudio.google.com/)
   - Generate a free API key
   - Paste it in the "Vault" settings on the Penguin Coder dashboard
   - Start generating personalized roadmaps!

## 📖 How to Use

### Search for a Career Path
```
1. Enter any job role (e.g., "Product Manager", "ML Engineer", "DevOps Architect")
2. View market data: avg salary, experience needed, demand level
3. Explore similar roles and career transitions
```

### Generate a Learning Roadmap
```
1. Select your target role
2. Choose your current experience level (Beginner/Intermediate/Advanced)
3. Click "Generate Roadmap"
4. Receive a custom 5-stage syllabus with resources & timelines
```

### Compare Career Options
```
1. Add multiple roles to the ROI Matrix
2. View side-by-side comparison
3. Sort by: Highest Salary, Fastest Learning Path, Best ROI
4. Make data-driven decisions
```

### Get AI Career Advice
```
1. Open the Chat Oracle
2. Ask questions about: career transitions, market trends, skill relevance
3. Get personalized, market-informed answers instantly
```
